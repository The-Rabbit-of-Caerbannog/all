#Bash it If you are on a Debian based Linux system (Kali, anyone?) and installing a requirements.txt is giving you trouble - try making your own bash script to do it for you.

  1. Replace the requirements variable with the ones you'd like to install, separated by spaces.
  2. Save in a yourfilename.txt file on your machine.
  3. Run bash yourfilename.txt 
#!/usr/bin/env bash

sudo apt-get update

sudo apt-get upgrade

sudo apt-get install python3

requirements="List Your Requirements Here"

sudo pip3 install $requirements
